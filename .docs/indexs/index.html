OpenTab

一个安全又好看的本地标签页

Edge 星愿 CRX 离线版

功能特性

一切为了效率与美感

新标签主页

顶部搜索栏支持 Bing / Google / Baidu 与自定义引擎。收藏网站网格展示，支持分类浏览。

个性化设置

自定义壁纸、图标大小、圆角调节，甚至侧边栏隐藏，一切随你心意。

数据导入导出

一键导出配置文件，跨设备轻松恢复。数据由你掌控。

快捷圆环

任意界面按住“修饰键 + 鼠标左键”呼出快捷圆环。最高支持 8 个常用页面跳转，让效率翻倍。

支持 Ctrl / Alt / Shift 触发键，圆环大小随心调节。

本地优先，隐私无忧

OpenTab 不依赖远程后端存储用户配置。主要数据保存在浏览器本地，
不会上传到任何服务器。但请记得随时备份你的配置文件。
